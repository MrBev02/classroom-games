Two-screen setup
  1. You're on the Host page — keep it on your laptop.
  2. Click Start Game — the Board opens in a new tab. Drag it to the projector and fullscreen it (F11).
  3. The dot above turns green when the Board is linked. Lost the tab? Re-open it with Open Board (top right).

Keep both windows in the same browser on the same computer. Running your own copy offline? Serve the folder first (e.g. python -m http.server 8080) and use the http://localhost:8080/… addresses — opening the files directly won't sync.
