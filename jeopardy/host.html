Two-screen setup
  1. You're on the host page — keep it on your laptop.
  2. Click Start Game — the Board opens in a new tab. Drag it to the projector and fullscreen it (F11).
  3. Lost the Board tab? Re-open it any time with Open Board (top right). Both screens stay in sync automatically.

Keep both windows in the same browser on the same computer. Running your own copy offline instead of the web link? Serve the folder first (e.g. python -m http.server 8080) and use the http://localhost:8080/… addresses — opening the files directly won't sync.
