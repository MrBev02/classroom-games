# Sounds

Drop an MP3 named **`think.mp3`** in this folder and the host's
**🎵 Think music** button will play it (e.g. the Jeopardy "Think!" track —
it's copyrighted so it isn't bundled here; source it yourself).

If `think.mp3` is missing, the game falls back to a built-in synthesised
"thinking clock" that ticks for ~30 seconds and ends with a time's-up chime.

The correct-answer ding and wrong-answer buzzer are synthesised in the
browser — no files needed.
