# Sounds

The host's **🎵 Think music** plays when you open a clue. There are two ways
to give it your own track (the Jeopardy "Think!" track is copyrighted, so none
is bundled — bring your own):

1. **Upload it on the host setup screen** (easiest — no files to edit). The
   track is kept in your browser and remembered for next time. Click **Remove**
   there to clear it.
2. **Drop an MP3 named `think.mp3` in this folder.** Used when nothing has been
   uploaded.

If you do neither, the game falls back to a built-in synthesised "thinking
clock" that ticks for ~30 seconds and ends with a time's-up chime.

The correct-answer ding and wrong-answer buzzer are synthesised in the
browser — no files needed.
